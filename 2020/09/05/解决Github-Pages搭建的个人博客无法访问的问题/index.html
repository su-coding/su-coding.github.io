解决Github Pages搭建的个人博客无法访问的问题
发表于2020-09-05|更新于2020-09-05
|字数总计:184|阅读时长:1分钟|阅读量:

前言

最近一段时间，发现使用Hexo+Github搭建的个人博客部署后无法访问

显示如下

  • 火狐浏览器

点击“接受风险后继续”，显示空白

  • 谷歌浏览器

问题原因

经常访问github 的小伙伴都知道，这个通常是域名解析的问题，DNS解析

于是ping一下这个地址：

发现DNS都被解析为127.0.0.1，怪不得无法访问，那么问题就很容易解决了。

解决办法

在https://www.ipaddress.com中查询对应的真实IP地址

复制其中一个真实IP地址

在C:\Windows\System32\drivers\etc目录的hosts文件中，添加配置：IP地址 域名

保存后，重新访问即可。
