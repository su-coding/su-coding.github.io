    一. 下载与安装Git

    1.下载

    ​ Git官网网址为: https://git-scm.com

    点击Downloads

    选择Windows版本

    选择64位版本

    2.安装

    ​ 一直点击Next即可.

    3.配置

    初始化
    • 配置Git用户名与邮箱
    git  config --global  user.name  'GitHub用户名'
git  config --global  user.email  'GitHub邮箱'
    参考文档

    https://www.cnblogs.com/ximiaomiao/p/7140456.html

    生成SSH Key并连接GitHub

    在 Git Bash中执行以下命令

    • 查看是否存在SSH key

    ​ 检查本机用户home目录中是否存在SSH Key，如果存在则跳到第三步

    $cd ~/.ssh
    • 创建SSH Key
    $ssh-keygen -t rsa -C "GitHub邮箱"

    ​ 提示输入SSH Key文件存储路径，直接回车（选择默认路径）

    Generating public/private rsa key pair.
Enter file in which to save the key (/c/Users/Lenovo/.ssh/id_rsa):

    ​ 提示输入密码，直接回车（空密码）

    Enter passphrase (empty for no passphrase):

    ​ 提示确认密码，直接回车

    Enter same passphrase again:

    ​ 提示密码生成信息

    • 在GitHub添加SSH Key

      • 拷贝SSH Key

        在Git Bash中执行命令:

      $clip < ~/.ssh/id_rsa.pub

      • 设置GitHub的SSH Key

        • 选择Settings
        • 选择SSH and GPG keys，点击New SSH Key

        • 在Key中粘贴SSH Key，并点击Add SSH Key

    • 测试SSH Key

      在Git Bash中执行命令

    $ssh -T git@github.com

    ​ 提示信息1:

    二. 下载与安装Node.js

    1.下载

    ​ Node.js官网网址为:https://nodejs.org

    选择LTS版本

    2.安装

    ​ 一直点击Next即可.

    3.测试

    ​ 打开Git Bash，输入node -v查看版本

    $node -v

    三. 初始化Hexo

    ​ 在Git Bash中执行命令

    • 安装淘宝镜像源

      $npm install -g cnpm --registry=https://registry.npm.taobao.org
      • 查看cnpm版本
      $cnpm -v

    • 安装Hexo

      $cnpm install -g hexo-cli
      • 查看Hexo版本
      $hexo -v

    • 新建blog文件夹，在文件夹中使用Git Bash

    • 初始化Hexo

      $hexo init
      • 初始化成功提示信息
      INFO  Cloning hexo-starter https://github.com/hexojs/hexo-starter.git
INFO  Install dependencies
added 183 packages from 420 contributors and audited 189 packages in 21.577s

8 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities

INFO  Start blogging with Hexo!

    • 开启Hexo

      $hexo s
$hexo server

      localhost:4000访问

    四. 部署Hexo到GitHub

    ​ 在blog文件夹下使用Git Bash

    • 安装hexo部署插件
    $cnpm install --save hexo-deployer-git

    • 在GitHub中创建Repository

      • 选择New repository
      • Repository name: GitHub用户名.github.io
      • Description: 可选
      • 点击Create repository

      • 拷贝SSH

    • 配置部署

      在blog根目录的_config.yml中配置如下信息

      deploy:
  type: git
  repo: [新建仓库的SSH]
  branch: master

    • 部署

      $hexo d
$hexo deploy

    • 测试
