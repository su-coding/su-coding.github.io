Hexo-Yilia主题配置

Yilia主题网址:https://github.com/litten/hexo-theme-yilia.git

设置基本信息

  • 头像
    • avatar: 头像图片路径
    • favicon: 网站图标路径

“所有文章”功能配置

  • 在blog根目录中执行如下命令:

    $npm i hexo-generator-json-content --save

  • 在blog根目录中_config.yml配置文件中添加如下配置

    jsonContent:
    meta: false
    pages: false
    posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
    excerpt: false
    categories: false
    tags: true

配置个人链接图片

  • 在/themes/yilia/source/中新建assets目录，并添加链接图片

  • 在/themes/yilia/_config.yml配置文件中

    • 在subnav中配置个人链接路径

      subnav:
  #github: "#"
  #weibo: "#"
  #rss: "#"
  #zhihu: "#"
  qq: "/assets/QQ.jpg"
  weixin: "/assets/WeChat.jpg"
  #jianshu: "#"
  #douban: "#"
  #segmentfault: "#"
  #bilibili: "#"
  #acfun: "#"
  mail: "mailto:824491225@qq.com"
  #facebook: "#"
  #google: "#"
  #twitter: "#"
  #linkedin: "#"

配置打赏

  • 在/themes/yilia/_config.yml配置文件中
    • reward_type
      • 0 关闭打赏
      • 1 reward:true属性的文章才有打赏
      • 2 所有文章均有打赏
    • reward_woding: 打赏提示语
    • alipay: 支付宝支付图片路径
    • weixin: 微信支付图片路径

在/themes/yilia/layout/_partial/left-col.ejs文件中，

标签后添加


    总文章数 <%=site.posts.length%>

显示文章摘要

在md格式的文章中添加 标签后面的内容不显示

文章显示目录

在themes/yilia/_config.yml中设置toc为2

添加归档

在themes/yilia/_config.yml中设置menu

menu:
  归档: /archives/index.html

文章插入网易云音乐

  • 获取音乐HTML代码

  • md格式文章中添加HTML代码

博客插入背景音乐

  • 获取音乐HTML代码

![image-20200818145954925](../../../Typora workplace/other/Hexo-Yilia主题配置/image-20200818145954925.png)

  • 在themes/yilia/layout/_partial/left-col.ejs文件中nav标签中添加如下代码:

    <% if (theme.music && theme.music.enable){ %>
    
    
        音乐HTML代码
    
    
<% } %>

  • 在themes/yilia/_config.yml中配置

    # 网易云音乐插件
music:
  enable: true # 是否开启音乐功能
  autoplay: true  # 是否开启自动播放

“微信分享”功能配置

在themes/yilia/layout/_partial/post/share.ejs文件中

在

var sUrl = url.replace(/index\.html$/, '');
sUrl = /^(http:|https:)\/\//.test(sUrl) ? sUrl : 'http:' + sUrl;

后面添加

test=sUrl.substring(20);

将

//pan.baidu.com/share/qrcode?url=' + sUrl

改为

//api.qrserver.com/v1/create-qr-code/?size=150x150&data=主页网址' + test

文章置顶

  • 安装插件
$npm uninstall hexo-generator-index --save
$npm install hexo-generator-index-pin-top --save
  • 在/themes/主题/layout/post.ejs中,最前面添加如下代码
<% if (page.top) { %>
  
  置顶
  |
<% } %>
  • 在文章的Front-matter中添加top: 值，值越大，优先级越高
---
title: 2019
date: 2019-02-14 16:10:03
top: 5
---

  • 在blog根目录_config.yml配置文件中

    • top
    • date

    属性值

    • 1 正序
    • -1 倒序

文章加密

  • 安装插件
$npm install hexo-blog-encrypt
  • 在blog根目录_config.yml配置文件中设置
encrypt:
    enable: true
  • 在文章的Front-matter中
    • password: 密码
    • abstract: 博客显示的摘要信息
    • message: 输入密码的提示信息
